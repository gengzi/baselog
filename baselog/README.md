# BaseLog
提供基础日志打印
（1）提供TraceId，明确每一个请求链路
（2）提供controller层，基础日志打印，包含请求信息和响应信息，提供自定义日志设置，提供自定义配置
（3）提供baselog注解，明确每个方法的入参和出参

## 未来计划
（1）提供特定报错，报警日志，发送邮件，手动关闭
（2）提供接口的幂等性校验，防止重复提交接口
（3）接入分布式链路追踪，逐步支持各种请求
